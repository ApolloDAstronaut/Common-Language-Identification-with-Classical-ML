This is intended as a learning project and attempts to implement a model to classify speech, i.e. identify what language is being spoken
without using deep learning.

My current approach involves a non-linear Support Vector Machine (The idea is to use a Radial Basis Function) to classify speech based on
the Mel-Frequency Cepstral Coefficients (MFCC). Multiple resources have stated that MFCC is sufficient to accurately model speech.
